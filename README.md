ТикТок мод, который работает в связке с ТикФинити. Полное взаимодействие со зрителями. Обработка событий лайков, присоединений, комментариев, подарков

Возможности:

- Появление бабочек с никами каждого зрителя, который присоединяется к трансляции. Исчезание через 10 секунд
- Появление бабочек с ником и текстом комментария (обрезанным до 27 символов) за сообщение в чат трансляции и дублирование сообщения в чат игры
- Появление противника - летающей рыбы с ником зрителя, который отправил подарок за одну и более монету
- Отхил игрока за каждые 10 лайков зрителей на 1 здоровье. Появляется дружественная стрекоза с ником, системой накопления и количеством лайков в формате +3❤️
- Формирование своих собственных логов со списком подписчиков, списокм зрителей, списков дарителей и списком модераторов, которые проявили активность на трансляции
- Добавлено отображение того, к какому стриму выполнено подключение
- Если заходит зритель, который уже есть влогах, то:
1. Если этот человек уже был подписан - спавнится красный слизень-защитник
2. Если этот человек модератор - спавнится огенный слизень-защитник
3. Если этот человек даритель - спавнится золотой слизень
4. Если этот человек поделился трансляцией - спавнится радужный слизень и даётся случайный баф
5. Если этот человек - новый подписчик, спавнится обычный синий слизень
- Защита от перенаполнения мира существами при большой активности на стримах (тестировалось на трансляции 23К+ зрителей)
- Уникальный ИИ для дружественных слизней
- Для каждого дружественного слизня свой таймер жизни. Оно отображается рядом с ником. Как только таймер истекает, слизень превращается в стрекозу с ником слизня, из которого превращалась
- Полный отказ от ТикФинити. Переход на собственный локальный вебсокет
- Полная интеграция с Твитч + ТикТок + Ютюб
- Асинхронное подключение ко всем трём платформам с полным логированием процесса в чат
- Проверка подключения к вебсокетам
- Независимая работа с доступными платформами. Если у вас, например, нет ТикТок, но есть Твитч и Ютюб, ничего страшного. Ошибок не будет
- За подарки спавнятся мобы в зависимости от того, в каком биоме находится игрок
- Добавлены тестовые команды в чат для проверки событий:
/tiktest gift        // симуляция подарка (1 раз)
/tiktest gift 5      // симуляция подарка x5
/tiktest share      // симуляция репоста
/tiktest sub        // симуляция подписки
/tiktest join sub   // вход подписчика
/tiktest join mod   // вход модератора
/tiktest join gift   // вход дарителя

В планах:
- Спавн более грозных противников за донаты 5+ монет. Чем больше стоимость подарка, тем серьёзнее противник
- Усиление противников, спавнящихся за монеты в зависимости от прогресса игры. Чем дальше пройдена игра, тем серьёзнее противники
- Возможность вызвать босса за счёт голосования в чате. Если наберётся 10+ "BOSS" - начинается битсва с боссом
